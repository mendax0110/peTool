\doxysubsection{Detailed Description}
\doxylink{interface_key_event_responder}{Key\+Event\+Responder} implements the \doxylink{class_n_s_text_input_client-p}{NSText\+Input\+Client} protocol as is required by the mac\+OS text input manager.

The mac\+OS text input manager is invoked by calling the interpret\+Key\+Events method from the key\+Down method. Keyboard events are then evaluated by the mac\+OS input manager and valid text input is passed back via the insert\+Text\+:replacement\+Range method.
